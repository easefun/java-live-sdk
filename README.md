# POLYV 直播 JAVA SDK ###

## 安装
通过 Maven:
```xml
<dependency>
  <groupId>net.polyv</groupId>
  <artifactId>java-live-sdk</artifactId>
  <version>1.7.2</version>
</dependency>
```

## 运行环境
JDK 6 及以上

## 使用方法
参考文档：[SDK文档](https://dev.polyv.net/2018/liveproduct/l-sdk/javalivesdk/)
